<div>
I wrote a code for wrap-around snake game in Ardunio. 
The four buttons are controlled by one analog pin. 
Which button is pressed is detected by the amount of current flowing to the analog pin, as the buttons are sequentially connected with resistors. The end state shows 'GG' and an arrow pointing at the reset button for another round of game. This version of the game doesn't have a concrete win state, but it can be easily added by modifying few lines of code.  
</div>
