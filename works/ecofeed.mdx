## ECOfeed is an app that encourages students on campus to reduce disposables, reuse reusables, and save the planet. ##


<br/>

Each time a student (user) uses their personal reusables, they could accumulate points (feeds) to win prizes like eco-friendly swags.
This application won the **Best Environmental Hack** at Tartan Hacks 2020.

<br/>

**Role:** UI/UX design, Front-end develop <br/>
**Duration: ** < 48hr <br/>
**Tools:** Android Studio, Java, Microsoft Azure <br/>
**Team: ** Ashley Kim, Maggie Huang
